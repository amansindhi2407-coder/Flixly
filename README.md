# Flixly
Flixy is a full-stack video streaming web application inspired by platforms like Netflix and Amazon Prime. It allows users to browse movies and series, search content in real time, and watch videos using a custom-built media player. The project includes features such as interactive UI components, autoplay hero banners, playback controls, and watch progress tracking. It focuses on delivering a smooth user experience with responsive design and dynamic content handling. Media files (images and videos) are not included in this repository due to size and copyright limitations.
